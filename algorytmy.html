ŚCIĄGA Z ALGORYTMÓW

1. ALGORYTM EUKLIDESA - NWD

Cel

Obliczanie największego wspólnego dzielnika dwóch liczb.

Implementacja

def nwd(a, b):
    while b:
        a, b = b, a % b
    return a

Jak to działa

  1. Jeśli b = 0, to NWD(a,b) = a
  2. W przeciwnym wypadku NWD(a,b) = NWD(b, a mod b)
  3. Powtarzamy aż b = 0

Złożoność

  • Czasowa: O(log(min(a,b)))
  • Pamięciowa: O(1)

Zastosowania

  • Skracanie ułamków
  • Obliczanie NWW (najmniejszej wspólnej wielokrotności): NWW(a,b) = (a*b)/NWD(a,b)
  • Kryptografia (algorytm RSA)

2. SITO ERATOSTENESA

Cel

Wyznaczanie wszystkich liczb pierwszych mniejszych lub równych n.

Implementacja

def sito(n):
    p = [True] * (n + 1)
    for i in range(2, int(n**0.5) + 1):
        if p[i]:
            for j in range(i*i, n + 1, i):
                p[j] = False
    return [i for i in range(2, n + 1) if p[i]]

Jak to działa

  1. Tworzymy tablicę p o rozmiarze n+1, początkowo wszystkie wartości są True
  2. Zaczynając od 2, dla każdej liczby pierwszej oznaczamy wszystkie jej wielokrotności jako nie-pierwsze
  3. Wystarczy sprawdzić liczby do pierwiastka z n
  4. Wynikiem są indeksy tablicy o wartości True

Złożoność

  • Czasowa: O(n log log n)
  • Pamięciowa: O(n)

Zastosowania

  • Wyznaczanie dużej liczby liczb pierwszych
  • Testowanie pierwszości liczb
  • Rozwiązywanie zadań kombinatorycznych

3. CIĄG FIBONACCIEGO (ITERACYJNIE)

Cel

Obliczanie n-tego wyrazu ciągu Fibonacciego.

Implementacja

def fib(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a

Jak to działa

  1. Zaczynamy od F(0)=0 i F(1)=1
  2. Kolejne wyrazy obliczamy jako sumę dwóch poprzednich
  3. Po n krokach otrzymujemy F(n)

Złożoność

  • Czasowa: O(n)
  • Pamięciowa: O(1)

Zastosowania

  • Analiza struktur naturalnych
  • Algorytmy optymalizacyjne
  • Problem podziału

4. SORTOWANIE

4a. SORTOWANIE BĄBELKOWE

Cel

Sortowanie tablicy przez porównywanie i zamianę sąsiednich elementów.

Implementacja

def bubble(t):
    n = len(t)
    for i in range(n):
        for j in range(n - 1 - i):
            if t[j] > t[j + 1]:
                t[j], t[j + 1] = t[j + 1], t[j]

Jak to działa

  1. Przechodzimy przez tablicę n razy
  2. W każdym przejściu porównujemy sąsiednie pary elementów
  3. Jeśli są w złej kolejności, zamieniamy je miejscami
  4. Po każdym przejściu największy element jest już na swoim miejscu

Złożoność

  • Czasowa: O(n²) - najgorszy i średni przypadek
  • Pamięciowa: O(1)

4b. SORTOWANIE PRZEZ WSTAWIANIE

Cel

Sortowanie tablicy przez wstawianie każdego elementu w odpowiednie miejsce.

Implementacja

def insertion(t):
    for i in range(1, len(t)):
        x = t[i]
        j = i - 1
        while j >= 0 and t[j] > x:
            t[j + 1] = t[j]
            j -= 1
        t[j + 1] = x

Jak to działa

  1. Zaczynamy od elementu t[1]
  2. Porównujemy go z elementami na lewo od niego
  3. Przesuwamy większe elementy w prawo
  4. Wstawiamy element w odpowiednie miejsce
  5. Powtarzamy dla każdego elementu

Złożoność

  • Czasowa: O(n²) - najgorszy i średni przypadek, O(n) - najlepszy przypadek
  • Pamięciowa: O(1)

4c. SZYBKIE SORTOWANIE (QUICKSORT)

Cel

Sortowanie tablicy za pomocą strategii "dziel i zwyciężaj".

Implementacja

def quick(t):
    if len(t) <= 1:
        return t
    p = t[0]
    l = [x for x in t[1:] if x < p]
    r = [x for x in t[1:] if x >= p]
    return quick(l) + [p] + quick(r)

Jak to działa

  1. Wybieramy element pivot (tutaj pierwszy element)
  2. Dzielimy tablicę na dwie części:
    • elementy mniejsze od pivota
    • elementy większe lub równe pivotowi
  3. Rekurencyjnie sortujemy obie części
  4. Łączymy posortowane części z pivotem

Złożoność

  • Czasowa: O(n log n) - średni przypadek, O(n²) - najgorszy przypadek
  • Pamięciowa: O(log n) - głębokość rekursji

5. DODATKOWE ALGORYTMY WARTE ZAPAMIĘTANIA

5a. WYSZUKIWANIE BINARNE

def binary_search(t, x):
    left, right = 0, len(t) - 1
    while left <= right:
        mid = (left + right) // 2
        if t[mid] == x:
            return mid
        elif t[mid] < x:
            left = mid + 1
        else:
            right = mid - 1
    return -1  # element nie znaleziony

5b. POTĘGOWANIE SZYBKIE

def power(a, n):
    if n == 0:
        return 1
    if n % 2 == 0:
        return power(a*a, n//2)
    else:
        return a * power(a, n-1)

5c. WYSZUKIWANIE ELEMENTU MAKSYMALNEGO/MINIMALNEGO

def find_max(t):
    maxval = t[0]
    maxidx = 0
    for i in range(1, len(t)):
        if t[i] > maxval:
            maxval = t[i]
            maxidx = i
    return maxidx, maxval

5d. LICZBY PIERWSZE - TEST PIERWSZOŚCI

def is_prime(n):
    if n <= 1:
        return False
    if n <= 3:
        return True
    if n % 2 == 0 or n % 3 == 0:
        return False
    i = 5
    while i * i <= n:
        if n % i == 0 or n % (i + 2) == 0:
            return False
        i += 6
    return True

5e. KONWERSJA LICZB MIĘDZY SYSTEMAMI

# Konwersja z systemu dziesiętnego na dowolny
def to_base(n, base):
    digits = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    result = ""
    while n > 0:
        result = digits[n % base] + result
        n //= base
    return result or "0"

# Konwersja z dowolnego systemu na dziesiętny
def from_base(s, base):
    digits = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    result = 0
    for digit in s:
        result = result * base + digits.index(digit)
    return result
