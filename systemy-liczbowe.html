1. Konwersja liczb w konsoli przeglądarki (JavaScript)

Konwersja liczby na dowolny system (2–36)

(255).toString(2);   // "11111111"  → binarny
(255).toString(16);  // "ff"        → szesnastkowy
(255).toString(5);   // "2010"      → piątkowy

Konwersja napisu w danym systemie na dziesiętny

parseInt("2010", 5);     // 255 → z systemu piątkowego
parseInt("ff", 16);      // 255 → z szesnastkowego
parseInt("11111111", 2); // 255 → z binarnego

Przykłady działań na liczbach w różnych systemach

(parseInt("2024", 5) + parseInt(1044,5)).toString(5); // "3123"
(parseInt("2024", 5) - parseInt(1044,5)).toString(5); // "430"

2. Python – przeliczanie liczby dziesiętnej na binarny (zgodne z wytycznymi maturalnymi)

def decimal_to_binary(n):
    wynik = 0       # Na początku wynik binarny to 0 (jeszcze nic nie policzyliśmy)
    mnoznik = 1     # Mnożnik pozwala nam "ustawić" kolejne bity na odpowiednim miejscu (1, 10, 100 itd.)

    while n > 0:    # Dopóki liczba n jest większa od zera – wykonujemy pętlę
        reszta = n % 2           # Dzielimy n przez 2 i zapisujemy resztę – to kolejny bit
        wynik = wynik + reszta * mnoznik  # Dodajemy ten bit na właściwe miejsce w liczbie binarnej
        mnoznik = mnoznik * 10   # Przesuwamy się w lewo (kolejne miejsce dziesiętne w binarnym zapisie)
        n = n // 2               # Dzielimy n przez 2 – przechodzimy do kolejnego bitu

    return wynik     # Zwracamy wynik zapisany jako liczba wyglądająca jak zapis binarny

3. Pseudokod – konwersja dziesiętna na binarną

Funkcja KONWERSJA_DZIESIETNA_NA_BINARNA(n)
    Ustaw wynik ← 0
    Ustaw mnoznik ← 1

    Dopóki n > 0 wykonuj:
        reszta ← n MOD 2
        wynik ← wynik + reszta × mnoznik
        mnoznik ← mnoznik × 10
        n ← n DIV 2
