ŚCIĄGA Z MICROSOFT EXCEL

1. FUNKCJE LOGICZNE I WARUNKOWE

JEŻELI

=JEŻELI(warunek; wartość_jeśli_prawda; wartość_jeśli_fałsz)

Przykład: =JEŻELI(A1>50;"Zdał";"Nie zdał")

LICZ.JEŻELI

=LICZ.JEŻELI(zakres; kryteria)

Przykład: =LICZ.JEŻELI(A1:A10;">50") - zlicza wartości większe od 50

LICZ.WARUNKI

=LICZ.WARUNKI(zakres1; kryteria1; [zakres2; kryteria2]...)

Przykład: =LICZ.WARUNKI(A1:A10;">50";B1:B10;"Warszawa") - zlicza wiersze, gdzie wartość w kolumnie A > 50 i wartość w kolumnie B to "Warszawa"

SUMA.JEŻELI

=SUMA.JEŻELI(zakres_kryteriów; kryteria; [zakres_sumowania])

Przykład: =SUMA.JEŻELI(B1:B10;"Warszawa";C1:C10) - sumuje wartości w kolumnie C, gdzie w kolumnie B jest "Warszawa"

SUMA.WARUNKÓW

=SUMA.WARUNKÓW(zakres_sumowania; zakres1; kryteria1; [zakres2; kryteria2]...)

Przykład: =SUMA.WARUNKÓW(C1:C10;A1:A10;">50";B1:B10;"Warszawa") - sumuje wartości w kolumnie C, gdzie A > 50 i B = "Warszawa"

2. FUNKCJE STATYSTYCZNE

  • =MAX(zakres) - największa wartość
  • =MIN(zakres) - najmniejsza wartość
  • =ŚREDNIA(zakres) - średnia arytmetyczna
  • =MEDIANA(zakres) - wartość środkowa
  • =LICZ(zakres) - liczba komórek zawierających liczby
  • =LICZ.PUSTE(zakres) - liczba pustych komórek
  • =LICZ.NIEPUSTE(zakres) - liczba niepustych komórek

3. WYSZUKIWANIE DANYCH

WYSZUKAJ.PIONOWO (VLOOKUP)

=WYSZUKAJ.PIONOWO(wartość_szukana; tabela; nr_kolumny; [typ_wyszukiwania])

Przykład: =WYSZUKAJ.PIONOWO(A1;$D$1:$F$10;2;FAŁSZ) - szuka wartości z A1 w pierwszej kolumnie zakresu D1:F10 i zwraca wartość z drugiej kolumny

INDEKS + PODAJ.POZYCJĘ

=INDEKS(tablica; PODAJ.POZYCJĘ(wartość_szukana; zakres_wyszukiwania; [typ_dopasowania]))

Przykład: =INDEKS(C1:C10;PODAJ.POZYCJĘ("Jan";A1:A10;0)) - zwraca wartość z kolumny C w wierszu, gdzie w kolumnie A jest "Jan"

4. IMPORTOWANIE I PRZETWARZANIE DANYCH

Import z CSV

  1. Zakładka Dane → Z tekstu
  2. Wybierz plik CSV
  3. Wybierz odpowiedni separator (przecinek, średnik)
  4. Ustaw typy danych w kolumnach
  5. Zakończ import

Filtrowanie

  1. Zaznacz zakres danych
  2. Dane → Filtr
  3. Kliknij strzałkę w nagłówku kolumny
  4. Wybierz kryteria filtrowania

Sortowanie

  1. Zaznacz zakres danych
  2. Dane → Sortuj
  3. Wybierz kolumnę i sposób sortowania (rosnąco/malejąco)

5. TWORZENIE WYKRESÓW

Podstawowe kroki

  1. Zaznacz dane
  2. Wstaw → Wykres → wybierz typ wykresu
  3. Dodaj tytuł wykresu: Narzędzia wykresów → Projekt → Dodaj element wykresu → Tytuł wykresu
  4. Dodaj etykiety osi: Narzędzia wykresów → Projekt → Dodaj element wykresu → Tytuły osi
  5. Dodaj etykiety danych: Narzędzia wykresów → Projekt → Dodaj element wykresu → Etykiety danych

6. TABELE PRZESTAWNE

Tworzenie tabeli przestawnej

  1. Zaznacz dane
  2. Wstaw → Tabela przestawna
  3. Wybierz miejsce umieszczenia tabeli
  4. Przeciągnij pola do odpowiednich obszarów:
    • Filtry raportu - filtrowanie całej tabeli
    • Etykiety wierszy - dane w wierszach
    • Etykiety kolumn - dane w kolumnach
    • Wartości - dane liczbowe do analizy

Aktualizacja tabeli przestawnej

Po zmianie danych źródłowych: kliknij prawym przyciskiem myszy na tabeli → Odśwież

7. PRAKTYCZNE SZTUCZKI

Adresowanie bezwzględne i względne

  • A1 - adres względny (zmienia się przy kopiowaniu)
  • $A$1 - adres bezwzględny (nie zmienia się przy kopiowaniu)
  • $A1 - kolumna bezwzględna, wiersz względny
  • A$1 - kolumna względna, wiersz bezwzględny

Autofiltry i sortowanie zaawansowane

  • Dane → Sortuj i filtruj → Sortowanie niestandardowe
  • Dane → Sortuj i filtruj → Filtr → Filtr zaawansowany

Funkcje zagnieżdżone

Przykład: =JEŻELI(A1>90;"Celujący";JEŻELI(A1>75;"Bardzo dobry";JEŻELI(A1>50;"Dobry";"Niedostateczny")))
